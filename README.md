# Smart-Irrigation-Energy-Prediction
A sustainable agriculture project using Machine Learning to predict irrigation energy consumption and optimize water and power usage for eco-friendly farming.
This project applies Machine Learning to promote sustainable farming by predicting the energy consumption of irrigation systems. It enables farmers to plan irrigation efficiently, minimizing both electricity usage and water waste.

Sustainability Impact

Reduces unnecessary power usage in agriculture.
Prevents over-irrigation and conserves water.
Decreases carbon footprint from excessive electricity.
Supports SDG 7 (Affordable & Clean Energy), SDG 12 (Responsible Consumption), and SDG 13 (Climate Action).
